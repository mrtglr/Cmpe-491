Project Description

Easy search is a service API that can store and process documents for the user to easily search through thousands of documents. Easy search can be used for applications and websites that has a lot of data and needs a way to find the documents according to their search. Performing such search operation needs a lot of preparation and setting up, easy search simplifies the operation into just uploading the documents and sending search requests.
